Acerca de este sitio

Este sitio ofrece guías sencillas para ayudarte a realizar gestiones en línea con el gobierno de Guatemala. NO TRABAJAMOS DIRECTAMENTE CON EL GOBIERNO NI SOMOS UNA ENTIDAD OFICIAL, solo brindamos instrucciones para que puedas hacer tus trámites fácilmente.
Al final de cada guía encontrarás los enlaces oficiales de las instituciones.
Si prefieres que nosotros hagamos la gestión por ti, cobramos Q25 POR CADA GESTIÓN REALIZADA, además del pago que exija la institución (por ejemplo, Q30 para penales o Q50 para policiacos). Para el NIT, que es gratuito, el costo es solo el servicio de Q25.

Elige la guia que necesites o conoce nuestro servicio para que te ayudemos a hacer tus tramites
